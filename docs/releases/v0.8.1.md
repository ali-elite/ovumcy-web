# Ovumcy v0.8.1

Release date: 2026-03-29

## Highlights

- Public patch release for the OIDC/SSO feature set introduced in `v0.8.0`.
- Removes the remaining CodeQL warnings from the local OIDC browser-test runtime helper before the public release tag.

## Security and quality

- The local OIDC test helper no longer reflects unvalidated callback values into rendered markup, no longer echoes internal error messages in JSON, and no longer accepts arbitrary `post_logout_redirect_uri` values.
- The release was revalidated with `npm run lint` and all three OIDC browser lanes on Chromium, Firefox, and WebKit.

## Upgrade notes

- No new database migration is introduced beyond the schema changes already shipped in `v0.8.0`. If you are upgrading directly from `v0.7.2` or earlier, Ovumcy will still apply schema version `016_oidc_logout_states`.
- Tagged images publish under `ghcr.io/ovumcy/ovumcy-web:v0.8.1`.
- Existing deployments can upgrade in place and pin `OVUMCY_IMAGE=ghcr.io/ovumcy/ovumcy-web:v0.8.1` if they do not want to track future tags manually.
